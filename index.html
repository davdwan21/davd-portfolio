"Bringing dreams into reality."

Creativity shouldn't be limited by hardware. Most modern rendering systems are designed for professionals and require strong GPUs. inVISION solves both of these problems, allowing any individual to run low-cost renders of any object in 3D. With a single photo of an object, inVISION generates a 3D mesh of the object and then displays it in a lightweight 3D renderer that the user can interact with through gesture controls.

Skills: Python, PyGame, NumPy, TripoSR, MediaPipe, OpenCV

"Can you see the concept?"

Audio transcription is unreliable — what if we back it with low-cost video transcription? Modern lip-reading systems are largely impractical for this use case due to the scale of the infrastructure, but Silent Speech solves this by simplifying the predict with facial landmarks. Silent Speech is a real-time lip-reader that detects spoken words from video alone using MediaPipe facial landmarks, a simple CNN, and a GRU Recurrent Neural Network.

Results: SanD Hacks 2026 | Best Overall Hack Winner

Skills: Python, PyTorch, MediaPipe, OpenCV

Links — Github | Devpost | Demo Video

"Predicting the future for the ocean's sake"

Government policy can have unpredictable effects on the environment, so we fixed this by simulating the effects of policy actions on the environment. TIDAL-WAVE is a Python-based ecosystem simulation project for the Southern California coastal environment. The simulator combines a policy parser, a rule-based ecosystem engine, and agent-style species behavior to model how environmental actions affect the California Current system.

Results: DataHacks 2026 | Best Use of NVIDIA Brev | Data Analytics Track 3rd Place
